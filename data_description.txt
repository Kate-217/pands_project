Data description:

The dataset contains 150 instances, each of which represents one of three classes of Iris flowers.

Each class contains 50 instances.

The names for the classes are Iris-virginica, Iris-setosa and Iris-versicolor.

The dataset stores the following information about the flowers: 

sepal length, sepal width, petal length and petal with. 
I will use the following attributes for this project:


"sep_length","sep_width","pet_length","pet_width","class"

The types of variables:

"sep_length","sep_width","pet_length","pet_width" - Continuous (Float), cm 
"class" - Categorical (String)
